- **天空渐变**: 每个季节独特的天空色彩变化

这是一个使用Python和Pygame创建的高性能季节循环月季花绽放动画，具有**大幅增强的粒子效果**和**丰富的季节元素**。动画展示了从春季绽放到冬季凋零的完整生命周期，配备华丽的季节特效和流畅的60FPS性能。

## 🚀 快速开始

### 环境要求
- Python 3.7+
- pygame 2.0+

### 安装步骤

1. **创建虚拟环境（推荐）**
   ```bash
   python -m venv .venv
   
   # Windows
   .venv\Scripts\activate
   
   # macOS/Linux  
   source .venv/bin/activate
   ```

2. **安装依赖**
   ```bash
   pip install pygame
   ```

### 运行动画

#### 方式1: 使用启动器
```bash
python launcher.py
```

#### 方式2: 直接运行
```bash
python rose_animation.py
```

## 📊 动画阶段

### 生命周期时序

1. **盛开期** (2秒) 🌸
   - 春季色调，天空蓝渐变背景
   - 樱花粉花瓣从30%展开到100%
   - 花瓣飞舞特效 + 春雨效果
   - 粒子爆发效果

2. **维持期** (3秒) 🌹
   - 夏季色调，金色渐变背景
   - 深粉红花瓣完全绽放
   - 萤火虫闪烁 + 阳光光斑
   - 脉动环境光照效果

3. **凋零期** (2秒) 🍂
   - 秋季色调，夕阳橙红渐变
   - 橙色花瓣开始飘落
   - 落叶飘摆 + 秋风条纹效果
   - 飘摆物理模拟

4. **完全凋零** (1秒) ❄️
   - 冬季色调，冷蓝白渐变
   - 钢蓝色花瓣消失
   - 雪花飞舞 + 十字雪花形状
   - 宁静冬日过渡

5. **重置期** (0.33秒) 🔄
   - 快速重新开始春季循环
   - 无缝循环过渡

## 🎨 技术实现

### 核心组件

#### `EnhancedPetal` - 花瓣类
- 8个控制点的平滑曲线
- 独立旋转和弯曲动画  
- 生命周期状态管理
- 飘落物理模拟

#### `EnhancedParticle` - 粒子类
- 多种粒子类型（闪光、发光、魔法、飘落花瓣）
- 物理运动模拟
- 生命周期管理
- 透明度渐变

#### `SeasonalRose` - 主花朵类
- 6层花瓣系统
- 季节颜色管理
- 粒子生成控制
- 发光效果渲染

#### `SeasonalBloomColors` - 颜色系统
- 四季颜色定义
- 颜色插值算法
- 动态背景变化

### 性能优化技术

- **对象池**: 重用粒子对象
- **层次渲染**: 按深度排序绘制
- **LOD系统**: 距离相关的细节层次
- **批量绘制**: 减少绘制调用
